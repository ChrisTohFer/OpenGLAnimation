Basic requirements for rendering:
-Vertex Array Object
    -Vertex Buffer Object
    -Attribute pointers
    -Element Buffer Object

-Shader Program
    -Vertex Shader
    -Fragment Shader

-Transformation Matrices
    -World Matrix       
    -View Matrix        (Camera)
    -Projection Matrix  (Camera)

Extras
-Textures (Requires VBO to have UV coordinates)
    -Various textures passed as uniforms:
        -Albedo
        -Diffuse
        -Others...

-Lighting (Requires VBO normals and uniform materials to work effectively)
    -Diffuse
    -Ambient
    -Spectral
    -Others...
